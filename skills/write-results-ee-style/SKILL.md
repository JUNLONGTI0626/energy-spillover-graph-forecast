# Goal
Write paper-ready empirical text from model outputs.

# Style
- concise
- non-AI-sounding
- no exaggerated claims
- no casual wording
- explain economic meaning first, metrics second

# Inputs
- tables from `outputs/tables/`
- figures from `outputs/figures/`
- writing rules from `docs/result_writing_guide.md`

# Outputs
- concise draft text
- optional subsection structure
